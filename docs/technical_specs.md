# Technical Specifications

## Frameworks

App will use the following :

- Angular version 20+ (ngIf, ngFor are deprecated)
- PixiJS version 8.13.2

## Targets

The main target is mobile phones under android.

The game will be displayed in portrait mode.

Typical phones are Samsung Galaxy S8+, Samsung Galaxy A56,A57,A71 or Pixel 7.

## Coding Rules

I want:

1. Clean code, without unecessary casts (like `as any`), prefer getters
2. Centralization of code : Do not redeclare enums or arrays already present in the code
3. Comments in english only
